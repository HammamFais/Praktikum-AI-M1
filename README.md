## 📖 Deskripsi
Repository ini berisi tugas M1 Praktikum Kecerdasan Buatan tentang **Representasi Pengetahuan** menggunakan bahasa pemrograman **Prolog (SWI-Prolog)**. Representasi pengetahuan adalah cara merepresentasikan fakta dan aturan agar dapat diproses oleh sistem kecerdasan buatan.

---

## 📂 Struktur File

| File | Soal | Deskripsi |
|------|------|-----------|
| `2.pl` | Soal 2 | Representasi fakta dan aturan (teman, membenci, bermusuhan) |
| `4.pl` | Soal 4 | Hierarki organisasi perusahaan (bawahan, atasan, anak buah rekursif) |
| `6.pl` | Soal 6 | Pohon silsilah keluarga (orangtua, saudara, paman, bibi, kakek, nenek, cucu) |
| `9.pl` | Soal 9 | Representasi pembuat buku dan program (penulis, penulisbuku) |

---

## 🧠 Konsep yang Dipelajari

- ✅ Representasi fakta (*facts*) dalam Prolog
- ✅ Pembuatan aturan (*rules*) menggunakan operator `:-`
- ✅ Negasi menggunakan operator `\+`
- ✅ Rekursi untuk menelusuri hierarki
- ✅ Backtracking untuk mencari semua solusi
- ✅ Relasi keluarga bertingkat
